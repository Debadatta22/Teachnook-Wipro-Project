# Teachnook-Wipro-Project
This repository contains my project developed during the Teachnook–Wipro collaborative internship. It features a fully functional front-end website for electric motor vehicles, enabling users to explore, view details, and book electric scooters, built as per mentor-assigned tasks.

---------

# ⚡ E-Motor Vehicle Frontend Website  
### 🚀 Teachnook – Wipro Internship Project

---

## 📌 Project Overview  
This repository contains my project developed during the Teachnook–Wipro collaborative internship, where I learned and applied core web development concepts.  

The project is a fully functional frontend website for electric motor vehicles that allows users to:  
- 🔍 Explore electric scooters  
- 📄 View detailed specifications  
- 🛒 Simulate booking functionality  
- 🎯 Interact with a clean, responsive UI  

This project was built as per mentor-assigned tasks to strengthen real-world frontend development skills.

---

## 🛠️ Tech Stack  

- **HTML5** → Structure of the website  
- **CSS3** → Styling and responsive design  
- **JavaScript** → Interactivity and dynamic behavior  
- **Node.js** → Development environment and project handling  

---

## ✨ Features  

- Modern and responsive UI design  
- Scooter listing with detailed descriptions  
- Interactive buttons and navigation  
- Booking simulation interface  
- Clean layout following industry practices  

---

## 🎯 Learning Outcomes  

During this internship project, I gained hands-on experience in:  

- Frontend development fundamentals  
- UI/UX design principles  
- Responsive web design  
- JavaScript DOM manipulation  
- Structuring real-world web projects
